# NexusOMS Monorepo

Enterprise-grade Order Management System with Automated Supply Chain Lifecycle.

## Project Structure
- **/backend**: Node.js API with Decision Engine & MongoDB/JSON storage.
- **/flutter**: Standardized Flutter Mobile App with 14 functional screens.
- **/docs**: System documentation and credentials.

## Deployment
- **Backend**: Deploy the `/backend` folder to Render/Heroku.
- **Frontend**: Standard Flutter build for iOS/Android.
