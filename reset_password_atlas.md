# Password Reset Guide (from Current Screen)

Bhai, screenshot ke hisaab se yeh karein:

1.  Left side bar mein sabse neeche jaaiye, wahan **"SECURITY"** section hai.
2.  Usme **"Database & Network Access"** par click karein.
3.  Ab **"Database Access"** tab select karein.
4.  Wahan `nexusadmin` user ke aage **"EDIT"** button hoga, uspar click karein.
5.  **"Edit Password"** select karke naya password daalein aur **"Update User"** kar dein.

Naya password mujhe yahan bhej dein, main migration khatam kar dunga! 🚀
